========================================
💌 LOVE LETTER README - CUSTOMIZATION GUIDE 💌
========================================

Thank you for using this interactive love letter template!

This guide will help you personalize everything for your special someone.

========================================
📝 HOW TO CUSTOMIZE
========================================

Look for the section in the code marked:

"==================== CUSTOMIZATION SECTION ===================="

Edit the CONFIG variables between the comments. Do NOT change anything outside this section unless you know what you're doing!

========================================
✏️ WHAT YOU CAN CHANGE
========================================

1. NAME (Line ~15)

2.    Change: name: "Nancy"
3.   To: name: "YourCrushName"

4.      This appears in:
5.     - Page title (browser tab)
6.    - Main heading "For [Name]"
      -    - Greeting "My Dearest [Name]"
       
           - 2. TAGLINE (Line ~18)
            
             3.    Change: tagline: "5FT+ OF PURE SUNSHINE"
             4.   To: tagline: "Anything you want!"
            
             5.      Examples:
             6.     - "The Most Beautiful Girl"
             7.    - "My Sunshine"
                   -    - "Taller Than My Problems"
                        -    - Leave blank: ""
                         
                             - 3. COMPLIMENTS LIST (Lines ~21-29)
                              
                               4.    This is the list of nice things shown. You can:
                               5.   - Edit existing ones
                                    -    - Add more (copy-paste the format)
                                         -    - Remove some
                                          
                                              -    Format for each item:
                                              -       { icon: "Emoji", text: "Your compliment here" }
                                          
                                              -      Example additions:
                                              -     { icon: "🎵", text: "Your laugh is my favorite sound" }
                                              -    { icon: "☕", text: "You make mornings better" }
                                              -       { icon: "🌙", text: "You shine brighter than the stars" }
                                          
                                              -   4. THE QUESTION (Line ~32)
                                               
                                                  5.    Change: question: "Will you be my chat partner?"
                                                  6.   To: Any question you want to ask!
                                               
                                                  7.      Examples:
                                                  8.     - "Will you go on a date with me?"
                                                  9.    - "Will you be my Valentine?"
                                                        -    - "Coffee this weekend?"
                                                         
                                                             - 5. BUTTON TEXTS (Lines ~35-36)
                                                              
                                                               6.    buttonYes: "Yes! 🎉"
                                                               7.   buttonNo: "No 🙈"
                                                              
                                                               8.      Customize these however you like!
                                                              
                                                               9.  6. SUCCESS MESSAGE (Lines ~39-40)
                                                                 
                                                                   7.    This appears when they click YES:
                                                                 
                                                                   8.   successMessage: "I knew you were the best!"
                                                                   9.      successSubMessage: "Let's chat forever! 💕"
                                                                 
                                                                   10.  ========================================
                                                                   11.  🎨 ADVANCED CUSTOMIZATION (Optional)
                                                                   12.  ========================================
                                                                 
                                                                   13.  Want to change colors? Look for CSS variables at the top:
                                                                 
                                                                   14.  :root {
                                                                   15.      --envelope-bg: #ffe4e1;      // Envelope main color
                                                                   16.      --envelope-dark: #ffb7c5;    // Envelope shadow color
                                                                   17.      --envelope-darker: #ff9eb5;  // Darker accents
                                                                   18.  }
                                                                 
                                                                   19.  Change these hex codes to any colors you like!
                                                                   20.  Use a color picker online to find hex codes.
                                                                 
                                                                   21.  ========================================
                                                                   22.  📱 HOW TO USE
                                                                   23.  ========================================
                                                                 
                                                                   24.  1. Open the HTML file in any text editor (Notepad, VS Code, etc.)
                                                                        2. 2. Find the CUSTOMIZATION SECTION
                                                                           3. 3. Edit the text between quotes
                                                                              4. 4. Save the file
                                                                                 5. 5. Open in any web browser to test
                                                                                    6. 6. Send the HTML file to your special someone!
                                                                                      
                                                                                       7. ========================================
                                                                                       8. 💡 TIPS
                                                                                       9. ========================================
                                                                                      
                                                                                       10. - Keep compliments genuine and personal
                                                                                           - - Don't make the list too long (7-10 items is perfect)
                                                                                             - - Use emojis that match their personality
                                                                                               - - Test on your phone before sending - it should scroll properly
                                                                                                 - - The "No" button runs away - it's a fun game!
                                                                                                  
                                                                                                   - ========================================
                                                                                                   - ⚠️ IMPORTANT
                                                                                                   - ========================================
                                                                                                  
                                                                                                   - - Keep the quotes " " around text
                                                                                                     - - Keep the commas , after each item
                                                                                                       - - Don't delete the curly braces { }
                                                                                                         - - Don't delete the square brackets [ ]
                                                                                                           - - If something breaks, check you didn't delete a comma or quote
